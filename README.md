# Screener App

## Structure du projet

- `src/` : fichiers JavaScript de l’application
- `data/` : fichiers JSON/CSV/TXT utilisés par l’application
- `css/` : styles
- `icons/` : assets d’icônes
- `cloudflare-workers/` : workers Cloudflare

## Versioning GitHub

1. Initialiser le dépôt Git : `git init`
2. Ajouter les fichiers : `git add .`
3. Faire le premier commit : `git commit -m "Initial commit"`
4. Créer un dépôt sur GitHub puis le lier : `git remote add origin <url-du-repo>`
5. Pousser : `git push -u origin main`
